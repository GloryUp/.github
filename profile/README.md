# GloryUp

> Um momento guiado com Deus todos os dias.

GloryUp é um aplicativo iOS focado em:
- Devocional diário
- Bíblia offline
- Jornada espiritual
- Anotações pessoais
- Constância com Deus

A proposta do projeto é transformar leitura bíblica em hábito diário através de uma experiência simples, guiada e offline-first.

---

# Visão do Produto

A experiência do GloryUp evolui da simples leitura para uma jornada espiritual estruturada:

- Ler
- Refletir
- Anotar
- Orar
- Continuar

Tudo pensado para funcionar mesmo offline, mantendo simplicidade, foco e constância.

# GloryUp 1.1

A versão atual consolida a base do aplicativo.

## Funcionalidades

### Bíblia Offline
- 66 livros disponíveis localmente
- Busca por trechos
- Favoritos
- Histórico
- Compartilhamento de versículos

### Devocional Diário
- Fluxo guiado
- Reflexões
- Registro espiritual

### Perfil e Histórico
- Linha do tempo devocional
- Estatísticas de constância
- Histórico pessoal

### Login Melhorado
- Login por usuário ou e-mail
- Cadastro com login automático
- Busca sem diferenciação de maiúsculas/minúsculas
